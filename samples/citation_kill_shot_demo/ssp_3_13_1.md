# SSP §3.13.1 — System and Communications Protection (SC.L2-3.13.1)

## Implementation narrative

Hardseal monitors, controls, and protects communications at the external
boundaries and key internal boundaries of the system. Boundary enforcement
follows the architectural constraints set in Policy-AC-001 v2.0 (access
control baseline) and the operational workflow described in Runbook-IR-102
v1.0 for boundary-incident response.

The external boundary is enforced by Fortinet FortiGate 100F running firmware
`7.4.3` at `fgt-ext-01.hardseal.example`. Ruleset is version-controlled in
Git at `infra/firewall/fgt_ext_01.cfg`, reviewed under PR `#501` merged
2026-01-29.
